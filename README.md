# Marketing Analytics with SQL

This repository contains SQL queries developed as part of a marketing analytics project using PostgreSQL.

## Project Overview

The project focuses on analyzing digital marketing campaign performance by writing SQL queries to generate business insights.

### Key Analyses
- Daily advertising spend analysis
- ROMI performance evaluation
- Weekly top-performing campaigns
- Monthly reach analysis
- Longest continuous ad set activity

## Tools
- SQL
- PostgreSQL
- DBeaver
